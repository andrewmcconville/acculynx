AccuLynx Coding Challenge

github.com/andrewmcconville/acculynx andrewmcconville.com/acculynx/stackoverflow

The Task

Create a website that consumes the Stackoverflow.com API for recent question activity and display information about them.

From the Recent Questions:

  1. Display the title of the question with the highest score
  2. Display the sum of the reputation for all users in that request
  3. Display a list of all the Recent Questions. Show:
    1. The user who asked the question
      1. Gravatar
      2. Most recent badge
      3. Number of badges the user had 6 months ago
        1. Display the badge totals grouped by rank
    2. Title
    3. Link to the actual stackoverflow question
    4. Number of answers submitted
    5. Total score

Upload your solution to a public repository on github.com and send us the link.

Bonus: Use git from the beginning to have a full commit history

What I Used

  • Angular: For code organization
  • ngAnimate: To animate routes
  • AngularUI Router: For nested states and routes
  • FastClick: To remove the 300ms delay on mobile
  • Stack Exchange API: For the data
  • Bouncejs: To generate complex CSS3 animations
  • Google Fonts: For the Titillium Web font
